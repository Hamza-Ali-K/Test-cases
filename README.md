# Test-cases
TC1 – Valid Login
The Subject Expert enters a correct username and password and successfully logs into the system.
TC2 – Invalid Login
The system displays an “Invalid Credentials” error message when an incorrect password is entered.
TC3 – View Dashboard
The logged-in Subject Expert clicks on the dashboard option and the dashboard information is displayed successfully.
TC4 – Subscribe/Add Subject
The Subject Expert adds or subscribes to a new subject, and it appears successfully in the subject list.
TC5 – Logout Function
The user clicks the logout button and is successfully redirected to the login page.
